Módulo PostgreSQL Sobrenatural

Uma solução inovadora para gerenciar sua infraestrutura na Magalu Cloud.

Características do Módulo

  • Provisionamento fácil e rápido de instâncias PostgreSQL
  • Configuração automática de backups e segurança
  • Suporte para múltiplas zonas de disponibilidade
  • Reutilização de módulos em diferentes projetos

Como Usar

module "my_postgres" {
  source         = "./postgres-module"
  instance_name  = "my-db-instance"
  instance_class = "db.t3.micro"
  storage_size   = 20
  database_name  = "mydatabase"
  db_user        = "admin"
  db_password    = "supersecretpassword"
  environment    = "dev"
}
